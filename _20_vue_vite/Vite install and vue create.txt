Para crear un proyecto con vite: npm create vite viteProject --template vue-ts

Con yarn: yarn create vite viteProject --template vue 

No mezclar con npm luego de iniciar el projecto

Instalar modulos:

npm install == yarn

Iniciar projeto

npm run serve == npm run dev == yarn run dev

Instalar paquetes

Router:

yarn add vue-router:Este no genera el encarpetado, solo instala


Uso del @ en las rutas, se debe configurar el vite.config.js
